# Space Flight

Simple ASCII-based game built with `curses` and `asyncio`.

```bash
  .   *                   :                *       +                +   .     .
                        :     +
   *                      *      +                                     .  *
           :                                      *                           +
         :               +     .                        : .  + .  +   .      .
               :          :              . +
                                        .'.  +                           *
  *             *                    :  |o|                                .
                                :      .'o'.                              +
   :         .*       +    . +       * |.-.|                 +
                                       '   '             +  *
          .  +        :  .              ( )              *      +
                      :        .     +   )  *
              +                       * ( )  :                          .
      *                    :   . :.                     .               +   : *
                  *           ++  :     .      +      .       :
           +                *         .                     +.            :
                +           :                      .     *        :          *
                                      :              +                          :
```

## Usage

Download or clone repository then

```bash
$ python main.py
```
